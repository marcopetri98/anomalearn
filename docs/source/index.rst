anomalearn documentation
========================

**Date**: Apr 20, 2023 **Version**: Pre-alpha 0.0.2

:ref:`anomalearn <homepage>` is an open source EUPL1.2-licensed library
providing modular and extensible tools for developing time series anomaly
detection models for the `Python <https://www.python.org/>`_ programming language.
